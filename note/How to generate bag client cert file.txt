1목support SSL
clientcertsigned.pem: Client certificate in PEM format.
clientcertkey.pem: Private key of client certificate in PEM format.  (not encrypted)

2목How to create kbgen.out which is used for compression/decompression
step1:enter \nettv_2k16_mtk\TPV_internal
step2: make keybag_clean -> make libkeybag -> make keybag
step3밐under /nettv_2k16_mtk/TPV_internal/mnettv_app/videostore_mw/tool밃you can find the file kbgen.out
step4밐in makefile밃it will copy kgben.out  into /nettv3/TPV_internal/mnettv_app/videostore_mw/tool/keys
cp -f /nettv3/TPV_internal/mnettv_app/videostore_mw/tool/kbgen.out /nettv3/TPV_internal/mnettv_app/videostore_mw/tool/keys

3목On platform we will use kbgen.out to decompress ClientCert.bag밃then get two pem files.
step1밐cd /nettv_2k16_mtk/mnettvK2k16/opera/data/opera_dir밃you can find ClientCert.bag
step2밐copy kbgen.out and ClientCert.bag into USB몪
step3밐plug usb into TV SET, and then cd /mnt/usb/sda1
step4: Decompress the file ClientCert.bag by kbgen.out
./kbgen.out -h
./kbgen.out -xn ClientCert.bag

sh-3.2# cd output/
sh-3.2# ls
clientcertkey.pem clientcert.p12  clientcertsigned.pem

4목In build server, create client.p12 file with two pem files.
 openssl pkcs12 -export -password "pass:" -in clientcertsigned.pem -inkey clientcertkey.pem -out client.p12

5목Rename client.p12 to clientcert.p12 and then creat .bag file
./kbgen.out -cn Client.bag ./output/clientcertkey.pem ./output/clientcert.p12 ./output/clientcertsigned.pem


6목rename Client.bag to ClientCert.bag,Then copy it into /nettv_2k16_mtk/mnettvK2k16/opera/data/opera_dir/
